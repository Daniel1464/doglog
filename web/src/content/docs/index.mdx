## Example usage

```java
// Import
import dev.doglog.DogLog;
import dev.doglog.DogLogOptions;

// Configure
DogLog.setOptions(new DogLogOptions().withNtPublish(true));

// Log
DogLog.log("Arm/Position", motor.getPosition().getValue());
```
